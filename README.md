# Face Emotion Detection System

## Overview
The Face Emotion Detection System is a computer vision-based project that detects and classifies human emotions from facial expressions in real time. It uses deep learning techniques, specifically Convolutional Neural Networks (CNNs), to identify emotions such as happy, sad, angry, surprised, and neutral.

This system can be applied in areas like human-computer interaction, mental health analysis, and smart surveillance systems.

---

## Features
- Real-time face detection using webcam
- Emotion classification using deep learning (CNN)
- Supports image and video input
- Multiple emotion categories detection
- Simple and scalable design
